# rust-demo

## Install rustup

https://rustup.rs/

```
$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

To uninstall,

```
$ rustup self uninstall
```

## Create a project

```
$ cargo new rust-demo
```

or

```
$ mkdir rust-demo
$ cd rust-demo
$ cargo init
```

## rustfmt

https://github.com/rust-lang/rustfmt

```
$ rustup component add rustfmt
```
